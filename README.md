## Description

A message app with http protocol



----------

# Getting started

## Installation

Clone the repository

    git clone https://github.com/lyon93/http-chat

Switch to the repo folder

    cd http-chat
    
-----
## Installation

```bash
$ npm install
```

## Running the app

### Running Locally
```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### Running with Docker 

    docker run -p3000:3000 lyon93/http-chat:1.0.0
 

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
